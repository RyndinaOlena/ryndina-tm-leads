# ryndina-tm-leads

## Опис
Цей проєкт є веб-застосунком, розробленим за допомогою React, SCSS і Vite.

## Технології
- Vite
- React
- SCSS
- npm

## Структура
- `src/` — код застосунку.
- `components/` — компоненти.
- `pages/` — статичні файли.
- `styles/` — SCSS файли для стилів.

## Встановлення
- npm install
- npm run dev
- npm run build
